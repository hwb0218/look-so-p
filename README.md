# LookSoPrt
> React + Typescript + Firebase Cosmetic E-commerce   
> https://www.looksoprt.store
> #### Project setup  
> **`yarn install`**
> #### Compiles and hot-reloads for development
> **`yarn run dev`**
> #### Compiles and minifies for production
> **`yarn run build`**
