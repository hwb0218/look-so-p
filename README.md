# LookSoPrt
> React + Typescript + Firebase Cosmetic E-commerce   
> https://www.looksoprt.store

`npm`
```Shell
$ git clone git@github.com:hwb0218/look-so-p.git
$ cd look-so-p
$ npm install
$ npm run dev
```  

`yarn`
```Shell
$ git clone git@github.com:hwb0218/look-so-p.git
$ cd look-so-p
$ yarn install
$ yarn run dev
```
